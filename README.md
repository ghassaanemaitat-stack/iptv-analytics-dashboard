# 📡 IPTV Analytics Dashboard

A real-time IPTV traffic analytics dashboard built with vanilla JavaScript — no backend needed.

## Features
- Live visitor counter (simulated real-time)
- Channel traffic breakdown with animated bars
- Country-level visitor feed
- Bounce rate & session metrics

## About
Built for the IPTV community to visualize streaming traffic patterns.

🌐 **Main site:** [YOURSITE.com](https://strongchannels.com.com)

- Vanilla HTML/CSS/JS
- GitHub Pages (free hosting)
